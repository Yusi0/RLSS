Step 1 — GitHub 연동 정보 암호화 (최초 1회 설정)
아래 폼에 GitHub 정보를 입력하고 [암호문 생성]을 눌러 나온 암호문을 admin.html 소스코드의 ENCRYPTED_GIT_CONFIG 변수에 붙여넣으면 모든 관리자가 추가 설정 없이 배포 가능합니다.
암호문이 여기에 생성됩니다...
적용 방법: 위 암호문을 복사한 뒤, admin.html 소스코드에서
const ENCRYPTED_GIT_CONFIG = ''; 부분의 따옴표 안에 붙여넣고 저장하세요.
Step 2 (대안) — 이 브라우저에만 임시 토큰 저장
암호문 내장이 완료되기 전까지 이 브라우저에만 GitHub 토큰을 임시로 저장해 사용할 수 있습니다. 브라우저 localStorage에만 저장되며 다른 사람에게는 공유되지 않습니다.
Step 3 (옵션) — 관리자 비밀번호 해시 생성기
새로운 관리자를 추가하거나 비밀번호를 변경하려면 SHA-256 해시가 필요합니다. 아래에 원하는 비밀번호를 입력하고 [해시 생성]을 눌러 나온 해시값을 admin.html 소스코드의 ADMIN_ACCOUNTS 객체에 추가하세요.
